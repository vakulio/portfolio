## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/vakulio/portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run project in developer mode
   ```sh
    npm run dev
   ```
